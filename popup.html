Connect Anki

First time

This extension connects automatically once AnkiConnect is installed and Anki is open.

  1. Open Anki Desktop.
  2. Go to Tools -> Add-ons -> Get Add-ons....
  3. Paste add-on code 2055492159.
  4. Restart Anki.
  5. Keep Anki open, then click Authorize / Retry.
